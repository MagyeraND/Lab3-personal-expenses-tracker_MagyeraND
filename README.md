# Lab3-personal-expenses-tracker_MagyeraND


 Run the Python program by typing python3 expenses-tracker.py or running in your terminal.
 Use the menu to check balance, add expenses, view or search expenses, or exit.
 The program creates balance.txt automatically and updates it whenever you add money or expenses.
 Expense files are saved as expenses_YYYY-MM-DD.txt old files are archived using the shell script archive_expenses.sh.
